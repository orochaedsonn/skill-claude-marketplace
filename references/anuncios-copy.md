## Fórmulas de título por plataforma

### Mercado Livre
**Fórmula base:**
`[Palavra-chave exata de busca] [Produto] [Variação/Cor/Tam] [Diferencial] [Especificação técnica]`

**Exemplos por nicho:**

Moda:
`Calça Jeans Feminina Skinny Cintura Alta Elasticada Tamanho Grande Plus Size`

Beleza:
`Sérum Facial Vitamina C Anti-idade Clareador Hidratante 30ml Pele Oleosa`

Fitness:
`Corda de Pular Profissional Speed Rolamento Ajustável Crossfit Musculação`

**Dicas ML:**
- Priorize os termos com maior volume de busca no começo do título
- Inclua variações de escrita (Ex: "anti-idade" e "antienvelhecimento")
- Não repita palavras — use o espaço para cobrir mais termos de busca

### Shopee
**Fórmula base:**
`[Produto] [Especificação] [Benefício] [Palavra-chave long-tail] [Variação]`

**Dicas Shopee:**
- Inclua termos de busca coloquiais (como as pessoas realmente digitam)
- Use vírgulas para separar variações: "Preto, Branco, Rosa"
- Repita a keyword principal 2x se couber naturalmente

## Estrutura de descrição completa

### Bloco 1 — Gancho (primeiras 2 linhas)
Abra com dor/desejo do comprador. Nunca comece com o nome do produto.

Exemplo Beleza:
`Cansada de produtos que prometem clarear e não funcionam? Este sérum de Vitamina C pura age nas primeiras 2 semanas — ou devolvemos seu dinheiro.`

Exemplo Fitness:
`Quer treinar mais forte sem depender de academia? Esta corda de pular profissional foi desenvolvida para quem leva o treino a sério.`

### Bloco 2 — Benefícios (bullets)
Máximo 6 bullets. Formato: emoji + benefício + prova/especificação

```
✅ [Benefício principal] — [por quê / como]
✅ [Benefício 2] — [diferencial]
✅ [Benefício 3] — [resultado esperado]
✅ [Benefício 4] — [detalhe técnico relevante]
✅ [Benefício 5] — [para quem é ideal]
✅ [Benefício 6] — [garantia / confiança]
```

### Bloco 3 — Especificações técnicas
Tabela simples ou lista. Inclua tudo que evita dúvida e reclamação:
- Material, dimensões, peso
- Composição (se aplicável)
- Compatibilidade / indicação de uso
- Conteúdo da embalagem

### Bloco 4 — Chamada para ação + confiança
```
🛒 Compre agora e receba em [X dias] com frete grátis para todo o Brasil.
⭐ [X] clientes satisfeitos — avaliação [nota] no [Mercado Livre/Shopee]
🔄 Troca garantida em até 30 dias sem burocracia
```

## Palavras proibidas por plataforma

### Mercado Livre
- Proibido: "original", "genuíno", "oficial" (sem ser marca autorizada)
- Proibido: nomes de marcas concorrentes no título
- Proibido: contato (telefone, email, WhatsApp) em qualquer campo

### Shopee
- Proibido: links externos
- Proibido: referências a outras plataformas (ML, Amazon)
- Evitar: palavras em caps lock excessivo

## Checklist de qualidade antes de publicar

- [ ] Título usa os 60+ primeiros caracteres com keywords de alto volume?
- [ ] Descrição abre com gancho (problema ou desejo)?
- [ ] Há pelo menos 5 bullets de benefício?
- [ ] Especificações técnicas estão completas?
- [ ] Não há dados de contato no anúncio?
- [ ] Imagem principal está no padrão da plataforma?
- [ ] Preço está competitivo com os top 5 do ranking?

## Otimização de anúncios existentes

Quando o usuário colar um anúncio para revisar, avalie:

1. **Title Score**: keywords presentes? Comprimento adequado? Diferencial claro?
2. **Description Score**: gancho? bullets? CTA? specs completas?
3. **Conversion killers**: o que pode estar afastando compradores?

Entregue: versão original + versão otimizada + lista de mudanças feitas.
