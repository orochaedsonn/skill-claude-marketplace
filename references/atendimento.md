# Atendimento e Reputação — R9 Marketplace

## Princípios de atendimento em marketplace

- **Velocidade é reputação**: responder em menos de 24h é obrigação; menos de 2h é diferencial
- **Tom**: sempre cordial, nunca defensivo, nunca confrontacional
- **Objetivo**: resolver o problema E preservar a avaliação
- **Proibido**: discutir com cliente publicamente, prometer o que não pode cumprir

---

## Templates de resposta por situação

### 1. Pergunta pré-venda (dúvida sobre o produto)
```
Olá, [Nome]! Tudo bem? 😊

[Resposta direta à dúvida — seja específico]

Qualquer outra dúvida, estou à disposição. Será um prazer atendê-lo!

Abraços,
[Nome da loja]
```

### 2. Reclamação de atraso no envio
```
Olá, [Nome]! Peço desculpas pelo transtorno com o prazo de entrega.

Verifiquei seu pedido e [situação real: está a caminho / foi enviado em XX / ocorreu XX].

O rastreamento atualizado é: [código]

[Se possível]: Faremos o possível para garantir que chegue o mais rápido possível. Caso tenha qualquer outra dúvida, estou aqui.

Obrigado pela compreensão! 🙏
```

### 3. Produto com defeito / não funcionou
```
Olá, [Nome]! Lamentamos muito saber disso.

Queremos resolver isso da melhor forma para você. Poderia nos enviar [foto/vídeo do problema] para que possamos analisar?

Assim que recebermos, processamos imediatamente a [troca / reembolso / solução que couber].

Nosso objetivo é que você fique 100% satisfeito. Conte conosco! ✅
```

### 4. Pedido de devolução / arrependimento
```
Olá, [Nome]! Claro, sem problemas.

Você tem até 7 dias corridos a partir do recebimento para exercer o direito de arrependimento (CDC Art. 49).

Por favor, [instrução de como proceder na plataforma]. Assim que recebermos o produto de volta, processamos o reembolso completo em até [prazo].

Qualquer dúvida, estou aqui! 😊
```

### 5. Avaliação negativa recebida
**Resposta pública (visível para todos):**
```
Olá, [Nome]! Agradecemos o retorno e pedimos desculpas pela experiência.

[Se o problema foi real]: Já identificamos o que aconteceu e estamos corrigindo. Gostaríamos de resolver sua situação — por favor, entre em contato pelo chat para que possamos ajudá-lo diretamente.

[Se houve mal-entendido]: Gostaríamos de entender melhor o que ocorreu. Entre em contato pelo chat — faremos questão de esclarecer e resolver.

Sua satisfação é nossa prioridade. 🙏
```
> ⚠️ Nunca discuta publicamente. A resposta é para futuros compradores, não para o cliente que reclamou.

### 6. Mediação / reclamação formal (ML e Shopee)
```
Prezado [Nome],

Lamentamos profundamente a situação relatada. Nosso compromisso é sempre entregar a melhor experiência.

[Descreva os fatos objetivamente, sem culpar o cliente]

Propomos a seguinte solução: [solução concreta — troca, reembolso parcial, cupom, etc.]

Ficamos no aguardo do seu retorno para encerrarmos isso da melhor forma. ✅
```

## Gestão de reputação proativa

### Solicitação de avaliação positiva
Envie mensagem APÓS confirmação de entrega:
```
Olá, [Nome]! Esperamos que seu [produto] tenha chegado perfeitinho! 😊

Se ficou satisfeito, sua avaliação faz uma diferença enorme para nós. Leva só 30 segundos e nos ajuda muito a continuar melhorando.

Qualquer dúvida sobre o produto, conte conosco! 🙏
```
> ⚠️ ML e Shopee proíbem oferecer desconto em troca de avaliação — nunca faça isso.

### Métricas de reputação a monitorar
| Métrica | Meta mínima | Meta ideal |
|---|---|---|
| Avaliação média | 4.0+ | 4.5+ |
| % reclamações | < 3% | < 1% |
| Tempo de resposta | < 24h | < 2h |
| % cancelamentos | < 3% | < 1% |
| Envios no prazo | > 95% | > 99% |

## Palavras que nunca usar no atendimento

- "Impossível" — substitua por "Vamos verificar o que podemos fazer"
- "Não é culpa nossa" — nunca transfira culpa, resolva
- "Você deveria ter lido" — jamais culpe o cliente
- "Não temos como" — sempre ofereça uma alternativa
- "Isso é política da plataforma" — o cliente não quer saber; resolva de outro jeito
